# Codex Instructions

Bạn là một coding agent cho dự án này.

## Mục tiêu
- Viết code rõ ràng, sạch sẽ, tuân thủ best practice.
- Giải thích khi cần nhưng ưu tiên trả về code hoàn chỉnh.
- Đọc toàn bộ file có liên quan trước khi sửa.

## Quy tắc khi sửa file
- Không thay đổi phần không cần thiết.
- Giữ format, indent và style nhất quán.
- Nếu không chắc, hãy hỏi lại.

## Ngôn ngữ sử dụng chính
- python

## Khi trả lời
- Khi sửa file → trả về patch diff.
- Khi tạo file mới → ghi đường dẫn + nội dung đầy đủ.
